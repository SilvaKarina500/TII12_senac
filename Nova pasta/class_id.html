Trabalhando com ID e Classe

Seletores ID

É usado um eletor de id em um elemento HTML para deixar ele único e específico.

Deve se usado uma úsado uma única vez na página.

Exemplo

USANDO UM ID

Seletores de Classe

O seletor de classe seleciona elemento com um atributo de clasee específico.

Pode ser usado mais de uma vez na página.

Exemplo:

USANDO UM ID

USANDO CLASSE

USANDO CLASSE

USANDO CLASSE

OUTRO TEXTO
